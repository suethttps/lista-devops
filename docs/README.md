# Documentação — Lista de Tarefas

Documentação técnica da aplicação **Lista de Tarefas**, um CRUD em
**Ruby on Rails 8** com banco **PostgreSQL**.

O objetivo destes documentos é responder rápido a duas perguntas:

1. **O que faz a lista de tarefas funcionar?**
2. **Onde está cada peça no código?**
